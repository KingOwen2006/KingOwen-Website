History of cannons

The first recorded use of a cannon in Europe was to defend the city of Algeciras during the siege of 1343-44. This was the start of the cannon era as it continued to evolve cannons where separated by 3 classes

The cannon was separated into 7 categories which are labelled in the image above.

  • Muzzle - The muzzle is a point of vulnerability as if there was too much pressure the cannon could peel like a banana as this is where the cannon ball is released from.
  • Chase - This is the thinnest part of the cannon and was also to help to improve accuracy.
  • First/Second reinforce - This helped reinforce the cannon so that it reduces the chance of peeling.
  • Vent - This was so that there was a way to provides an external opening to ignite the propellant charge of gunpowder inside the barre
  • Base ring - This was a primary attachment point for a breeching loop
  • Cascable - This part was also for a attachment.

18th century cannons could also be made out of both iron and bronze they both had advantages and disadvantages depending on what it was used for. Iron ones are more common this is because it was lighter and less expensive however, it was more vulnerable to the elements and rusted easily.

There was different types of colours that cannons could be in this could because of the country however, most common colour was black.

Positives and negatives of rendering using cycles and Evee

Evee: Is a real time rendering engine and is faster than cycles so people can use this to get a rough idea on what it could look like in cycles. This is because it is much more lighter than Cycles. But it is not the best for photorealistic renders. This is because it is super inaccurate which often times ruin some renders. Evee also does not support all shading nodes therefore, it is also another limiting factor.

Cycles: This provides the photorealistic renders which allows a lot better accuracy this is because it uses path based raytracing. It isn’t the best for performance and is very heavy.

In conclusion you can use Evee for a more cartoon scene and realism for cycles.
